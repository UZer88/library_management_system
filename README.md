# Library Management System

Объектно-ориентированная система управления библиотекой на Python.

## Возможности

- 📚 Управление разными типами медиа:
  - Книги (физические и электронные)
  - Журналы
  - Аудиокниги
  - DVD
- 👥 Два типа пользователей: обычный (до 3 книг) и премиум (до 10)
- ⏱️ Разные сроки выдачи и штрафы для каждого типа медиа
- 📊 Расчёт штрафов за просрочку
- 🔍 Поиск по названию и автору
- 📈 Отслеживание популярности (счётчик выдач)

## Запуск

```bash
python library.py
```

## Технологии
- Python 3.8+
- ООП (наследование, полиморфизм, инкапсуляция, абстрактные классы)
